العيادة الطبية — نسخة HTML / CSS / JS
=====================================

الملفات:
  - index.html   : صفحة تسجيل الدخول (البريد الإلكتروني + كلمة المرور)
  - clinic.html  : صفحة العيادة الطبية (محمية — لا تفتح إلا بعد الدخول)
  - styles.css   : التنسيقات
  - app.js       : إعدادات الوصول (البريد المسموح + كلمة المرور المجزأة)

بيانات الدخول الافتراضية:
  البريد الإلكتروني : karimaismail1998@gmail.com
  كلمة المرور      : clinic2026

لتغيير كلمة المرور:
  1) افتح أي صفحة من التطبيق في المتصفح
  2) افتح Console (F12)
  3) نفّذ الأمر:   await sha256("كلمة_المرور_الجديدة")
  4) انسخ الناتج (سلسلة hex) وضعه داخل app.js في المتغيّر ALLOWED_PASSWORD_SHA256

طريقة التشغيل:
  - افتح index.html في المتصفح مباشرة، أو
  - ارفع كل الملفات كما هي إلى أي استضافة ثابتة (GitHub Pages, Netlify, ...).
